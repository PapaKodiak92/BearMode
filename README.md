# BearMode

**Enter BearMode. Get back on mission.**

BearMode is a daily discipline app with a no-BS bear coach named Kodiak. Users pick their daily mission, run focus blocks, recover when they drift, and stack real-life wins.

## MVP Features

- Daily mission board
- Side quests
- Habit tracking with Full / Minimum / Recovery wins
- Lock-In focus timer
- Drift Reset flow
- Kodiak coach messages
- Test roar + browser notification alert
- Local persistence through `localStorage`

## Tech Stack

- React
- Vite
- TypeScript
- Plain CSS for fast iteration

## Run Locally

```bash
npm install
npm run dev
```

Then open the local Vite URL in your browser.

## Build

```bash
npm run build
```

## Product Pitch

BearMode helps people stop drifting and get back on mission. It is not a boring todo app. It is a daily discipline system powered by a badass bear coach that reminds users to lock in, shrink the task, and keep stacking wins.
