# BearMode Product Brief

## One-Liner

BearMode is a daily mission app with a no-BS bear coach that helps users focus, recover when they drift, and stack real-life wins.

## Core Loop

1. User opens app in the morning.
2. Kodiak asks: “What’s the mission today?”
3. User chooses one main mission, side quests, and habits.
4. User starts Lock-In focus sessions.
5. When distracted, user taps “I’m Drifting.”
6. Kodiak helps shrink the task into a 5-minute comeback.
7. User stacks wins and reviews progress.

## Mascot

**Kodiak** is the bear coach. He should be tough, funny, and direct.

Voice examples:

- “ROOOAR. What’s the mission today?”
- “One task. No wandering. Move.”
- “You slipped. Doesn’t matter. Shrink the task and restart.”
- “Win stacked.”

## Notification Identity

BearMode notifications should be memorable and optionally hilarious:

- Soft nudge: bear grunt
- Mission alert: Kodiak roar
- Drift reset: short growl
- Victory: happy bear chuff

## Planes

### Free

- One main mission
- Limited side quests
- Basic habits
- Basic focus timer
- Limited history

### Plus

- Unlimited missions and habits
- Progress history
- Drift reset tracking
- Weekly review
- More coach personalities

### Pro

- AI daily planning
- AI weekly review
- Custom routines
- Accountability rooms
- Before/after journey timeline
